# Applied AI & LLM Engineering Lab

Welcome to my personal Applied LLM Engineering sandbox. This repository serves as a living document of my self-directed learning, hands-on practice, and project prototyping. 

My focus here is bridging the gap between theoretical Deep Learning concepts and practical, production-ready AI applications. The codebase spans from core algorithmic implementations to building full-stack LLM architectures.

## Focus Areas & Projects

This repository is organized into distinct domains based on the technologies and architectures I am currently exploring and mastering:

### 1. Applied LLMs & Conversational AI
Building the interface and application layer for Large Language Models.
* **Multi-Modal Chatbots:** Designing conversational agents capable of handling text, image, and audio inputs using API integrations.
* **UI Development:** Rapid prototyping of AI tools using `Gradio` for clean, interactive user interfaces.
* **AI Agents:** Implementing autonomous agents that can reason and execute multi-step tasks.

### 2. Retrieval-Augmented Generation (RAG) & Vector Systems
Moving beyond basic context windows to give models long-term memory and proprietary knowledge.
* **Vector Databases:** Setting up and managing vector stores for semantic search.
* **Advanced RAG Architectures:** Building systems that retrieve, rerank, and synthesize information to drastically reduce AI hallucinations.

### 3. Open-Source AI & Model Fine-Tuning
Working directly with weights and customized training loops rather than just API calls.
* **HuggingFace Ecosystem:** Leveraging the `transformers` library for automated, open-source GenAI solutions.
* **Model Evaluation:** Benchmarking and evaluating both frontier and open-source models for specific business and code-generation tasks.
* **Fine-Tuning:** Adapting open-source foundation models to compete with proprietary frontier models on specialized tasks.
